# ✅ Tasks — Modern To-Do

A sleek, modern to-do web app built with **vanilla HTML, CSS & JavaScript**. No frameworks, no build step.

## Features

- 🎨 Modern glassmorphism UI with animated aurora background
- 🌙 Light / dark theme (auto-detects system preference, remembers your choice)
- 📱 Fully optimized for **iPhone 15** — respects the Dynamic Island & home-indicator safe areas
- 💻 Responsive layout that looks great on laptop / desktop browsers
- 📊 Live progress bar and task counters
- 🔍 Filter by All / Active / Done
- ✏️ Double-click a task to edit it inline
- 💾 Saved automatically in your browser (localStorage)
- ♿ Accessible & respects reduced-motion preferences

## Run locally

Just open `index.html` in any browser — no server required.

## Tech

`HTML` · `CSS` · `JavaScript` — deployed on **GitHub Pages**.
